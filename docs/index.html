Quick Start

Get up and running in under a minute

1

Install the binary

# macOS / Linux chmod +x surfmate.io-* sudo mv surfmate.io-* /usr/local/bin/surfmate.io # Windows: Move to a folder in your PATH
2

Add to Claude Desktop

// macOS: ~/Library/Application Support/Claude/claude_desktop_config.json // Windows: %APPDATA%\Claude\claude_desktop_config.json { "mcpServers": { "surfmate": { "command": "surfmate.io" } } }
3

Start browsing

# Restart Claude Desktop, then try: "Go to news.ycombinator.com and summarize the top 5 stories" "Navigate to my email and wait for me to log in" "Take a screenshot of the current page"
