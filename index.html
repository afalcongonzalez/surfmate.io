Setup

Choose your AI assistant

1

Add to Claude config

Open Claude ‚Üí Settings ‚Üí Developer ‚Üí Edit Config and paste:

claude_desktop_config.json
{ "mcpServers": { "surfmate": { "command": "surfmate.io" } } }
2

Restart Claude and try it

"Open the browser and go to news.ycombinator.com"
"Open a browser and search Wikipedia for AI"
1

Add the MCP server

Run this command in your terminal:

Terminal
claude mcp add surfmate surfmate.io
2

Start using it

"Open the browser and go to github.com"
"Open a browser and search for React documentation"
1

Open MCP settings

Go to Cursor Settings ‚Üí MCP ‚Üí Add new global MCP server

Or edit the config file directly:

Mac ~/.cursor/mcp.json
Windows %USERPROFILE%\.cursor\mcp.json
2

Add the config

mcp.json
{ "mcpServers": { "surfmate": { "command": "surfmate.io" } } }
3

Restart Cursor and try it

"Open the browser and go to stackoverflow.com"
"Open a browser and search npm for axios"
1

Install ngrok

ngrok creates a public URL for your local server so ChatGPT can reach it.

macOS (Homebrew)
brew install ngrok

Or download from ngrok.com/download for other platforms.

Then sign up at ngrok.com (free) and run:

Terminal
ngrok config add-authtoken YOUR_AUTH_TOKEN
2

Start surfmate and ngrok

Open two terminal windows:

Terminal 1 - Start surfmate
surfmate.io -http -port 8080
Terminal 2 - Start ngrok
ngrok http 8080

ngrok will show a URL like https://abc123.ngrok.io ‚Äî copy this.

3

Add to ChatGPT

Go to ChatGPT ‚Üí Create a GPT ‚Üí Configure ‚Üí Add actions ‚Üí Import from URL:

Paste your ngrok URL + /openapi.yaml
https://abc123.ngrok.io/openapi.yaml

Replace abc123.ngrok.io with your actual ngrok URL. Save and start chatting!

Keep both terminals running while using ChatGPT. The browser opens on your computer so you can handle captchas.
